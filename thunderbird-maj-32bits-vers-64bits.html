Thunderbird MAJ 32bits vers 64bits : profil réinitialisé...

by yobi
published on 15 septembre 2019

Je viens de faire la mise à jour de Thunderbird 60 32 bits vers Thunderbird 68 en 64 bits sous windows 10. Et bim ! C'est le drame : le logiciel redémarre comme neuf... Tous mes paramétrages ont disparu.

Bref, je rassure l'auditoire tout est bien là mais il faut indiquer à Thunderbird quel profil par défaut utiliser. Pour ce faire, voici la marche à suivre :

  1. Cliquez sur le "?" dans les menus déroulants de Thunderbird.
  2. Cliquez sur "Information de dépannage"
  3. Dans la section "Paramètres de base de l'application", cliquez sur "about:profiles" à la ligne "Profils"
  4. A priori, deux profils s'affichent : le nouveau (Profil : default-release) qui est vierge et l'ancien (Profil : default) provenant de l'ancienne installation. Il suffit donc de définir l'ancien comme profil par défaut en cliquant sur le bouton "Définir comme profil par défaut"

Voilà, enjoy, tout rentre dans l'ordre, l'ancien profil est de retour.

Vous pouvez en profiter pour désinstaller la version 32bits.
