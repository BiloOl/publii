Windows & MTP

by yobi
published on 10 avril 2021

Le truc reloud : impossible d'avoir accès à l'espace de stockage de mon androphone sous TWRP et sous Lineage depuis l'explorateur fichiers de Windows. Le pilote ne s'installe pas depuis le gestionnaire de périphériques. 

Bref, la marche à suivre pour installer le pilote standard de Microsoft est la suivante :

  1. Connecter l'androphone au PC via un câble USB
  2. Ouvrir le gestionnaire de périphériques
  3. Dans l'onglet "Affichage" → "Afficher les périphériques cachés"
  4. Clic droit sur le "Périphérique inconnu" → "Propriétés"
  5. Sur l'onglet "Pilote" → "Mettre à jour le pilote"
  6. "Rechercher le pilote sur mon ordinateur"
  7. "Choisir parmi une liste de périphériques sur mon ordinateur"
  8. Clique sur "Appareil Mobile" → dans la nouvelle fenêtre choisir "Microsoft" dans l'encart de gauche et "Périphérique USB MTP" dans l'encart de droite

Vive la simplicité...
